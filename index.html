ASP.NET MVC Route Localizer

MvcRouteLocalizer is a free and open source tool of mine that is able to add localization support for URLs in an ASP.NET MVC application. This library brings an important functionality to the framework that's somehow always been missing, but now you can add URL localizations to your ASP.NET MVC application without changing the routing configuration.

You can also create lowercase-and-dashed URLs automatically from your action names. For example:

https://www.example.com/Company/AboutUs

becomes

https://www.example.com/company/about-us

just by adding a few lines of extra code. Although currently it only supports the old ASP.NET MVC framework with convention-based routing system, I'm planning to extend its functionality in the future.
